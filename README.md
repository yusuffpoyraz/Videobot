# 🤖 Universal Media Downloader Telegram Bot

A high-performance Telegram bot built with **Python**, **python-telegram-bot**, and **yt-dlp** that allows users to download video and audio from 1000+ social media platforms including YouTube, Instagram, TikTok, and Twitter.

## ✨ Features
- **Multi-Platform Support**: Works with almost any social media URL.
- **Format Selection**: Choose between Video (MP4) or Audio (MP3).
- **Quality Control**: Select specific resolutions (1080p, 720p, 480p).
- **Asynchronous Execution**: High responsiveness even during heavy downloads.
- **Auto-Cleanup**: Automatically deletes files from the server after sending to save storage.

## 🚀 Technical Stack
- **Language:** Python 3.10+
- **API Wrapper:** `python-telegram-bot`
- **Engine:** `yt-dlp` (The most advanced fork of youtube-dl)
- **Media Processing:** `FFmpeg`
